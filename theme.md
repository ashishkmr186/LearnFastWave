# Learnfast Premium Study Dashboard Theme

## Direction
Learnfast should feel like a premium desktop SaaS study assistant: cozy, modern, minimal, spacious, and calm. The quality bar is closer to Linear, Notion, Arc Browser, and Raycast than a busy education dashboard.

## Color Palette
- App background: `#F8F9FC`
- Sidebar: `#FFFFFF`
- Cards and toolbar: `#FFFFFF`
- Border: `#E8EAF2`
- Primary: `#6C63FF`
- Secondary: `#8B80F9`
- Success: `#56C271`
- Warning: `#F6B73C`
- Danger: `#FF6B81`
- Text primary: `#1F2937`
- Text secondary: `#6B7280`

Use soft pastel icon backgrounds and subtle gradients. Avoid bright saturated colors and heavy contrast unless needed for focus.

## Typography
- Font: Inter, with a system sans fallback.
- Main heading: `32px`, bold.
- Section title: `18px`, semibold.
- Body: `15px`.
- Caption: `13px`.
- Keep letter spacing normal except small uppercase sidebar group labels.

## Layout
- Desktop-first dashboard.
- Sidebar width: `280px`.
- Bottom toolbar height: `110px`.
- Rounded corners: `18px` to `24px`; toolbar can use `28px`.
- Use spacious page padding, soft shadows, and quiet borders.

## Component Guidelines
- Use Lucide icons only.
- Every icon should sit inside a soft colorful circular or rounded background.
- Sidebar groups should use labels such as `RECENT NOTES`, `PREVIOUS NOTES`, `FREE TRIAL`, and `SETTINGS`.
- Main content should remain calm and mostly empty until the user starts a task.
- Empty state illustration should be soft and friendly: open book, coffee, sparkles, and small plants.
- Bottom toolbar actions should lift slightly on hover, scale to `1.03`, and rotate icons by `5deg` over `200ms ease`.
